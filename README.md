really interesting data and thoughts about current ai trends. Here some insights I got from the presentation and the discussion.

- the developer role is evolving and ai can be a lever for productivity
- agent skills are the new alternative for instruction files
- openspec is the better alternative compared to speckit
- Using more tools to ensure ai generated Code quality and Security. For example SonarQube, ESLint, Ruff(Python Linter)
- use prompt library to get good working prompts and instructions
